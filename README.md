# sunny-blog

Next.js 15 (App Router) + TypeScript 기반 개인 기술 블로그

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Visualization**: D3.js 그래프 뷰
- **Search**: Voyage AI + Supabase pgvector RAG
- **AI**: Claude API 콘텐츠 자동 생성
- **Deploy**: Vercel

ZANNABI LAB의 AI Native 워크플로우 PoC 프로젝트
